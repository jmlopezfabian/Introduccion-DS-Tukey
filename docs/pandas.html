Pandas

Notebook notebooks/pandas.ipynb

Introducción

Pandas es la librería de Python para manipulación y análisis de datos tabulares. Permite trabajar con tablas que tienen columnas de distintos tipos —números, texto, fechas— y con etiquetas que tienen significado propio. Sus dos estructuras principales son la Serie (una columna) y el DataFrame (una tabla completa), y ofrece herramientas para seleccionar, filtrar, transformar y agregar datos de forma eficiente.

Series

Una Serie es la estructura de datos unidimensional de pandas. Piensa en ella como un array de NumPy al que se le agrega un índice: cada valor tiene una etiqueta asociada.

Desde una lista

Si no especificas un índice, pandas asigna enteros comenzando desde 0 — igual que NumPy, pero ahora con nombre.

import pandas as pd

calificaciones = pd.Series([88, 72, 95, 60, 83])
print(calificaciones)
# 0    88
# 1    72
# 2    95
# 3    60
# 4    83
# dtype: int64

La columna izquierda es el índice; la derecha, los valores. El tipo de dato aparece al final.

Desde un diccionario

Cuando creas una Serie desde un diccionario, las llaves se convierten automáticamente en el índice.

calificaciones = pd.Series({
    "Ana":    88,
    "Luis":   72,
    "Mía":    95,
    "Carlos": 60,
    "Sara":   83,
})
print(calificaciones)
# Ana       88
# Luis      72
# Mía       95
# Carlos    60
# Sara      83
# dtype: int64

# Acceder por etiqueta
print(calificaciones["Ana"])           # 88
print(calificaciones[["Ana", "Mía"]]) # Ana 88, Mía 95

El índice (Index)

El índice es la etiqueta que pandas asocia a cada fila. No es una columna más: es la clave de identidad de cada elemento. Por defecto es un rango entero (0, 1, 2…), pero puede ser cualquier valor: nombres, fechas, categorías.

¿Para qué sirve el índice?

El índice permite acceder a filas por etiqueta (no por posición), alinear automáticamente Series de distintos tamaños al operar entre ellas, y combinar tablas usando una clave común — similar a un JOIN en SQL.

Ejemplo Índice personalizado y alineación automática

Cuando sumas dos Series con índices distintos, pandas alinea por etiqueta. Las etiquetas que no coinciden producen NaN.

a = pd.Series([10, 20, 30], index=["x", "y", "z"])
b = pd.Series([1,  2,  3],  index=["y", "z", "w"])

print(a + b)
# w     NaN
# x     NaN
# y    22.0
# z    33.0
# dtype: float64

Solo y y z están en ambas Series; x y w no tienen par y producen NaN.

Para inspeccionar y manipular el índice:

s = pd.Series([88, 72, 95], index=["Ana", "Luis", "Mía"])

print(s.index)    # Index(['Ana', 'Luis', 'Mía'], dtype='object')
print(s.values)   # array([88, 72, 95])

# Reiniciar al índice entero por defecto
s_reset = s.reset_index(drop=True)
print(s_reset)
# 0    88
# 1    72
# 2    95
# dtype: int64

Métodos de Series

Las Series tienen métodos para inspección estadística, transformación elemento a elemento y manejo de valores nulos.

Ejemplo Inspección y estadísticas 
notas = pd.Series([88, 72, 95, 60, 83, 72, 88])

print(notas.describe())
# count     7.000000
# mean     79.714286
# std      12.638...
# min      60.000000
# 25%      72.000000
# 50%      83.000000
# 75%      88.000000
# max      95.000000

print(notas.value_counts())
# 88    2
# 72    2
# 95    1
# 83    1
# 60    1
# dtype: int64

print(notas.sort_values())                   # ascendente
print(notas.sort_values(ascending=False))    # descendente
Ejemplo Transformación con map

map aplica una función o un diccionario a cada elemento de la Serie.

notas = pd.Series([88, 72, 95, 60, 83])

# map con función lambda
letras = notas.map(lambda x: "A" if x >= 90 else ("B" if x >= 80 else "C"))
print(letras)
# 0    B
# 1    C
# 2    A
# 3    C
# 4    B
# dtype: object

# map con diccionario — útil para recodificar valores categóricos
grupos = pd.Series(["A", "B", "A", "B", "A"])
codigos = grupos.map({"A": 1, "B": 2})
print(codigos)
# 0    1
# 1    2
# 2    1
# 3    2
# 4    1
# dtype: int64
Ejemplo Manejo de valores nulos 
import numpy as np

s = pd.Series([88, np.nan, 95, np.nan, 83])

print(s.isna())
# 0    False
# 1     True
# 2    False
# 3     True
# 4    False

print(s.isna().sum())     # 2  → cantidad de nulos

# Rellenar nulos con la media

# 0    88.000000
# 1    88.666667
# 2    95.000000
# 3    88.666667
# 4    83.000000

# Eliminar elementos nulos

# 2    95.0
# 4    83.0

DataFrame

Un DataFrame es la estructura bidimensional de pandas: una tabla con filas y columnas. Cada columna es una Serie y todas comparten el mismo índice. Internamente, es un diccionario ordenado de Series.

datos = {
    "nombre":       ["Ana", "Luis", "Mía", "Carlos", "Sara"],
    "calificacion": [88,    72,     95,    60,       83   ],
    "horas":        [5.0,   3.5,    7.0,   2.0,      4.5  ],
    "grupo":        ["A",   "B",    "A",   "B",      "A"  ],
}

df = pd.DataFrame(datos)
print(df)
#    nombre  calificacion  horas grupo
# 0     Ana            88    5.0     A
# 1    Luis            72    3.5     B
# 2     Mía            95    7.0     A
# 3  Carlos            60    2.0     B
# 4    Sara            83    4.5     A
Ejemplo Cargar datos desde archivo

En la práctica, los DataFrames casi siempre se cargan desde archivos externos:

# Desde un CSV
df = pd.read_csv("data/csvs/examenes.csv")

# Ver las primeras filas para confirmar que cargó bien
print(df.head())

Selección de columnas

La forma en que seleccionas columnas determina qué tipo de objeto obtienes de regreso. La diferencia importa porque Series y DataFrames no tienen los mismos métodos disponibles.

Una columna vs múltiples columnas

df["columna"] (un corchete) devuelve una Serie. df[["col1", "col2"]] (doble corchete, lista adentro) devuelve un DataFrame.

# Una columna → Serie
notas = df["calificacion"]
print(type(notas))       # <class 'pandas.core.series.Series'>
print(notas.mean())      # 79.6

# Múltiples columnas → DataFrame
subconjunto = df[["nombre", "calificacion"]]
print(type(subconjunto)) # <class 'pandas.core.frame.DataFrame'>
print(subconjunto)
#    nombre  calificacion
# 0     Ana            88
# 1    Luis            72
# 2     Mía            95
# 3  Carlos            60
# 4    Sara            83

Filtrado de filas

Para seleccionar filas que cumplan una condición se usan máscaras booleanas: una Serie de True/False del mismo largo que el DataFrame. Las filas marcadas con True se conservan.

Máscara booleana

Una condición como df["calificacion"] >= 80 no filtra por sí sola — produce una Serie de booleanos. El filtrado ocurre cuando esa máscara se pasa dentro de df[...].

# Construir la máscara
mascara = df["calificacion"] >= 80
print(mascara)
# 0     True
# 1    False
# 2     True
# 3    False
# 4     True
# Name: calificacion, dtype: bool

# Aplicarla para filtrar
aprobados = df[mascara]
print(aprobados)
#   nombre  calificacion  horas grupo
# 0    Ana            88    5.0     A
# 2    Mía            95    7.0     A
# 4   Sara            83    4.5     A

# Forma compacta, en una línea
aprobados = df[df["calificacion"] >= 80]
Ejemplo Combinar condiciones con &, | y ~

Usa & (y), | (o) y ~ (no) para combinar condiciones. Cada condición individual debe ir entre paréntesis.

# Grupo A AND calificación >= 80
df[(df["grupo"] == "A") & (df["calificacion"] >= 80)]
#   nombre  calificacion  horas grupo
# 0    Ana            88    5.0     A
# 2    Mía            95    7.0     A
# 4   Sara            83    4.5     A

# Grupo A OR calificación >= 90
df[(df["grupo"] == "A") | (df["calificacion"] >= 90)]
#   nombre  calificacion  horas grupo
# 0    Ana            88    5.0     A
# 2    Mía            95    7.0     A
# 4   Sara            83    4.5     A

# Negar con ~: estudiantes que NO son del grupo A
df[~(df["grupo"] == "A")]
#    nombre  calificacion  horas grupo
# 1    Luis            72    3.5     B
# 3  Carlos            60    2.0     B
Profundización Por qué no usar and / or de Python

Los operadores and, or y not de Python operan sobre un solo valor booleano. Una máscara de pandas es una Serie de cientos o miles de booleanos, y Python no sabe cuál usar como representante de toda la Serie. El resultado es:

ValueError: The truth value of a Series is ambiguous.

Los operadores &, | y ~ son operadores bit a bit que pandas ha sobrecargado para operar elemento por elemento sobre Series — exactamente lo que necesitas al filtrar.

loc e iloc

Pandas ofrece dos indexadores especializados para seleccionar filas y columnas simultáneamente. La diferencia fundamental es en qué lenguaje le hablas a pandas para identificar lo que quieres.

.loc — por etiqueta

.loc[filas, columnas] usa etiquetas: nombres de índice y nombres de columnas. El slice con : es inclusivo en ambos extremos, a diferencia del slice de Python nativo.

df_named = df.set_index("nombre")   # nombres como índice de fila

# Una fila → devuelve Serie
print(df_named.loc["Ana"])
# calificacion    88
# horas            5.0
# grupo            A

# Varias filas
print(df_named.loc[["Ana", "Mía"]])

# Una celda específica
print(df_named.loc["Ana", "calificacion"])   # 88

# Slice de filas: "Luis" hasta "Carlos" (ambos incluidos)
print(df_named.loc["Luis":"Carlos"])

# Condición booleana — equivalente al filtrado con máscara
print(df_named.loc[df_named["calificacion"] >= 80])

.iloc — por posición entera

.iloc[filas, columnas] usa posiciones enteras (0, 1, 2…), ignorando por completo las etiquetas. El slice es exclusivo en el extremo derecho, igual que Python nativo.

# Primera fila (posición 0)
print(df.iloc[0])

# Filas en posición 0 y 2
print(df.iloc[[0, 2]])

# Filas 1 y 2 (posición 3 excluida)
print(df.iloc[1:3])

# Fila 0, columna 1 (calificacion)
print(df.iloc[0, 1])     # 88

# Últimas 2 filas, primeras 3 columnas
print(df.iloc[-2:, :3])
Ejemplo Comparación directa loc vs iloc 
Operación                  loc (etiqueta)                iloc (posición)
------------------------------------------------------------------------
Una fila              df_named.loc["Ana"]           df.iloc[0]
Una celda             df_named.loc["Ana","calif"]   df.iloc[0, 1]
Rango de filas        df_named.loc["Ana":"Mía"]     df.iloc[0:3]  (3 excluido)
Varias columnas       df_named.loc[:, ["a","b"]]    df.iloc[:, [0, 1]]
Condición booleana    df_named.loc[df["x"] > 5]     No aplica

Regla práctica: usa loc cuando tus índices tienen nombres significativos (fechas, IDs, nombres). Usa iloc cuando necesitas posiciones numéricas exactas, sin importar las etiquetas.

GroupBy

Imagina que tienes una tabla de estudiantes y quieres el promedio de calificación por grupo. GroupBy hace exactamente eso: divide la tabla en subconjuntos según los valores de una columna, aplica una función a cada subconjunto, y combina los resultados.

Split → Apply → Combine

Split: el DataFrame se divide en subgrupos según los valores únicos de la columna elegida. Apply: se calcula una función (media, suma, conteo…) dentro de cada subgrupo de forma independiente. Combine: los resultados de todos los grupos se unen en un nuevo DataFrame.

# Promedio de calificación por grupo
print(df.groupby("grupo")["calificacion"].mean())
# grupo
# A    88.666667
# B    66.000000
# Name: calificacion, dtype: float64

# Conteo de estudiantes por grupo
print(df.groupby("grupo")["nombre"].count())
# grupo
# A    3
# B    2
# Name: nombre, dtype: int64

# Múltiples métricas a la vez
print(df.groupby("grupo")["calificacion"].agg(["mean", "min", "max"]))
#        mean  min  max
# grupo
# A     88.67   83   95
# B     66.00   60   72
Ejemplo agg con nombres personalizados

Puedes nombrar cada métrica resultante usando la sintaxis nombre_col = ("columna", "función") dentro de agg().

resumen = df.groupby("grupo").agg(
    promedio_nota  = ("calificacion", "mean"),
    nota_maxima    = ("calificacion", "max"),
    horas_promedio = ("horas",        "mean"),
    n_estudiantes  = ("nombre",       "count"),
)
print(resumen)
#        promedio_nota  nota_maxima  horas_promedio  n_estudiantes
# grupo
# A          88.666667           95        5.500000              3
# B          66.000000           72        2.750000              2
Profundización GroupBy por múltiples columnas y reset_index

Puedes agrupar por más de una columna pasando una lista. El resultado tiene un MultiIndex en las filas; usa reset_index() para convertirlo en columnas normales.

datos2 = {
    "nombre":       ["Ana", "Luis", "Mía", "Carlos", "Sara", "Juan"],
    "calificacion": [88,    72,     95,    60,       83,     77   ],
    "grupo":        ["A",   "B",    "A",   "B",      "A",    "A"  ],
    "turno":        ["M",   "M",    "T",   "T",      "M",    "T"  ],
}
df2 = pd.DataFrame(datos2)

resultado = df2.groupby(["grupo", "turno"])["calificacion"].mean()
print(resultado)
# grupo  turno
# A      M        85.5
#        T        89.0
# B      M        72.0
#        T        60.0
# Name: calificacion, dtype: float64

# Aplanar el MultiIndex en columnas normales
resultado = resultado.reset_index()
print(resultado)
#   grupo turno  calificacion
# 0     A     M          85.5
# 1     A     T          89.0
# 2     B     M          72.0
# 3     B     T          60.0
